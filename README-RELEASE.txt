Boat Game Portable Release (macOS + Windows)
============================================

Quick start (no terminal commands needed)
-----------------------------------------
1) Install Node.js LTS from https://nodejs.org
2) Download the correct release asset:
   - macOS: boat-game-macos.zip
   - Windows: boat-game-windows.zip
3) Extract the zip.
4) Start with the launcher file:
   - macOS: Boat Game Launcher.command
   - Windows: Boat Game Launcher.bat
5) In the launcher menu:
   - Option 1 starts local play
   - Option 2 starts local play and creates a LocalTunnel share link

Notes
-----
- If macOS warns about opening scripts from the internet, allow it in Security settings.
- Keep the launcher terminal window open while playing.
- Press Ctrl+C in that window to stop the server/tunnel.
